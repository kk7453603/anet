# Документация ANet

ANet — инструмент для организации приватного, защищённого VPN-соединения на основе собственного транспортного протокола ASTP (ANet Secure Transport Protocol).

## Содержание

### Общие сведения

- [Протокол ASTP](protocol.md) — принципы работы протокола, процесс установления соединения, используемые алгоритмы шифрования

### Установка и настройка

- [Настройка сервера](server-setup.md) — пошаговое руководство по развёртыванию VPN-сервера
- [Настройка клиента](client-setup.md) — подключение к серверу на различных платформах (Linux, Windows, macOS, Android)
- [Сервер авторизации](auth-server.md) — развёртывание и настройка HTTP-сервиса `anet-auth` для управления пользователями

### Расширенная настройка

- [Раздельное туннелирование](split-tunneling.md) — настройка маршрутизации трафика (режимы включения и исключения)
- [Маскировка трафика](stealth.md) — параметры padding и jitter для противодействия DPI
- [Настройка QUIC-транспорта](quic-tuning.md) — оптимизация производительности соединения

## Совместимость версий

Клиент и сервер должны использовать совместимые версии протокола. Совместимость определяется вторым числом в номере версии: например, все версии `v0.5.x` совместимы между собой.
